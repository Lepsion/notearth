***Д*рево** — это количество сессий, которые были отыграны за данного персонажа. С ростом ***д*рева** персонаж становится сильнее:


| Сессия | Получает              |
|:------:|:--------------------- |
|  *1*   | Создание персонажа    |
| **2**  | +1 **[[НеКартонка/Характеристики/СИР\|СИР]]**        |
| **3**  | +[[НеКартонка/Механики/НеКартонка\|НеКартонка]]       |
| **4**  | +[[НеКартонка/Характеристики/Актив\|Актив]]            |
|  *5*   | Получение [[НеКартонка/Характеристики/Черта\|Черта]]   |
| **6**  | +1 **[[НеКартонка/Характеристики/СИР\|СИР]]**        |
| **7**  | +1 **[[НеКартонка/Характеристики/СИР\|СИР]]**        |
| **8**  | +[[НеКартонка/Механики/НеКартонка\|НеКартонка]]       |
| **9**  | +[[НеКартонка/Характеристики/Актив\|Актив]]            |
|  *10*  | Развитие [[НеКартонка/Характеристики/Черта\|Черта]]    |
| **11** | +1 **[[НеКартонка/Характеристики/СИР\|СИР]]**        |
| **12** | +1 **[[НеКартонка/Характеристики/СИР\|СИР]]**        | 
| **13** | +[[НеКартонка/Механики/НеКартонка\|НеКартонка]]       |
| **14** | +[[НеКартонка/Характеристики/Актив\|Актив]]            |
|  *15*  | Пробуждение [[НеКартонка/Характеристики/Черта\|Черта]] |

**Базовое значение** - 1, даёт 10 стартовых [[НеКартонка/Характеристики/СИР\|СИР]]-а.

***Т*очность** - это характеристика боевого персонажа. Данный показатель определяет кол-во кубов кидаемое для совершения точной атаки. Тогда сложность такой атаки будет определять ***л*овкость** цели. Если сноровкой обладает не игровой персонаж, тогда точность определяет [[НеКартонка/Механики/Характеристики/Сложность\|Сложность]].

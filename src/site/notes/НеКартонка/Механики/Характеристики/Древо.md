***Д*рево** — это количество сессий, которые были отыграны за данного персонажа. С ростом ***д*рева** персонаж становится сильнее:


| Сессия | Получает              |
|:------:|:--------------------- |
|  *1*   | Создание персонажа    |
| **2**  | +1 **[[НеКартонка/Механики/Характеристики/СИР\|СИР]]**        |
| **3**  | +[[НеКартонка/Механики/Характеристики/НеКартонка\|НеКартонка]]       |
| **4**  | +[[НеКартонка/Механики/Характеристики/Актив\|Актив]]            |
|  *5*   | Получение [[НеКартонка/Механики/Характеристики/Черта\|Черта]]   |
| **6**  | +1 **[[НеКартонка/Механики/Характеристики/СИР\|СИР]]**        |
| **7**  | +1 **[[НеКартонка/Механики/Характеристики/СИР\|СИР]]**        |
| **8**  | +[[НеКартонка/Механики/Характеристики/НеКартонка\|НеКартонка]]       |
| **9**  | +[[НеКартонка/Механики/Характеристики/Актив\|Актив]]            |
|  *10*  | Развитие [[НеКартонка/Механики/Характеристики/Черта\|Черта]]    |
| **11** | +1 **[[НеКартонка/Механики/Характеристики/СИР\|СИР]]**        |
| **12** | +1 **[[НеКартонка/Механики/Характеристики/СИР\|СИР]]**        | 
| **13** | +[[НеКартонка/Механики/Характеристики/НеКартонка\|НеКартонка]]       |
| **14** | +[[НеКартонка/Механики/Характеристики/Актив\|Актив]]            |
|  *15*  | Пробуждение [[НеКартонка/Механики/Характеристики/Черта\|Черта]] |

**Базовое значение** - 1, даёт 10 стартовых [[НеКартонка/Механики/Характеристики/СИР\|СИР]]-а.

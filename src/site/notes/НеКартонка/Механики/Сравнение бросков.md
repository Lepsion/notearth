#### Условные обозначения:
- 🔴 **Красный круг:** Вероятность 2d6+Бонус **ниже**, чем N кубиков (отрицательная разница)
- 🟡 **Жёлтый круг:** Вероятности равны или разница ±1% (нейтральный результат)
- 🟢 **Зелёный круг:** Вероятность 2d6+Бонус **выше**, чем N кубиков (положительная разница)

#### Ключ сравнения:
- **Строки:** 7+ → 3+, 8+ → 4+, 9+ → 5+, 10+ → 6+
- **Столбцы:** Бонус +0 → 1 кубик, +1 → 2 кубика, +2 → 3 кубика, +3 → 4 кубика, +4 → 5 кубиков
- **Значения:** Разница в процентах = (Вероятность 2d6+Бонус) - (Вероятность N кубиков)


#### Таблица требуемых бросков по характеристикам (порог автоматизма ≥90%)
**Правило:** Если вероятность успеха ≥90% — проверка считается автоматически пройденной (бросок не требуется). В таблице указаны проверки, которые **требуют броска**.

| Характеристика (число кубиков) | Проверки, требующие броска (сложность)       |
| ------------------------------ | -------------------------------------------- |
| **1**                          | все (простая, обычная, сложная, невероятная) |
| **2**                  | обычная (4+), сложная (5+), невероятная (6+) |
| **3** - **4**                  | сложная (5+), невероятная (6+)               |
| **5+**                          | невероятная (6+) |

**Пояснение:**
- **Простая проверка (3+)** становится автоматической при **характеристике ≥3**
- **Обычная проверка (4+)** становится автоматической при **характеристике ≥5**
- **Сложная проверка (5+)** становится автоматической при **характеристике ≥6**
- **Невероятная проверка (6+)** не становится автоматической в пределах характеристики ≤8 (требует броска всегда)

**Примечание:** Для невероятных действий (6+) порог 90% достигается только при характеристике около 15 кубиков.

**Пояснение к таблице:**  
Вероятность "провала" рассчитывается как шанс того, что количество успешных результатов (≥ указанного значения) **строго меньше** количества выпавших единиц (⚀).  

**Наблюдения:**
- Для **лёгких задач (3+)** вероятность такого провала уменьшается с ростом числа кубиков.
- Для **трудных задач (6+)** вероятность провала растёт с увеличением количества кубиков, достигая максимума (~37% при 6 кубиках).
- При одном кубике вероятность всегда равна 1/6 (16.67%), так как провал происходит только если выпадает единица.
